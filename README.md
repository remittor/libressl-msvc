# LibreSSL for Windows (MSVC)

Up-to-date LibreSSL sources: https://github.com/EtchDroid/LibreSSL

Just run the appropriate CMD-file to start building.

Out files:
```
.\bin\$(Platform)_$(Configuration)\
    openssl.exe

.\lib\$(Platform)_$(Configuration)\
    libcrypto.lib
    libssl.lib
    libtls.lib
```
